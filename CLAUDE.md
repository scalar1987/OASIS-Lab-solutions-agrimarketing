# 오아시스Vision 프로젝트

## 프로젝트 개요
AI Hub 식물 병 유발 통합 데이터를 활용한 컴퓨터 비전 프로젝트

## 프로젝트 구조
```
오아시스Vision/
├── AI Hub_Images/          # AI Hub 데이터셋 이미지
└── CLAUDE.md               # 프로젝트 컨텍스트 파일
```

## 데이터셋 정보
- **출처**: AI Hub - 104. 식물_병_유발_통합_데이터
- **데이터 유형**: 이미지 데이터 (Validation 원천데이터)
- **용도**: 식물 병해 탐지 및 분류

## 개발 환경
- **플랫폼**: Windows + WSL2 Ubuntu
- **언어**: Python (예상)
- **주요 라이브러리**:
  - OpenCV / PIL (이미지 처리)
  - TensorFlow / PyTorch (딥러닝)
  - NumPy, Pandas (데이터 처리)

## 코딩 스타일 가이드
- Python PEP 8 준수
- 한글 주석 사용 가능
- 함수/클래스명은 영문 사용

## 주의사항
- 대용량 이미지 데이터 처리 시 메모리 관리 필요
- 파일 경로에 한글이 포함되어 있으므로 인코딩 주의 (UTF-8 사용)
- WSL2 환경에서 Windows 경로 접근 시 `/mnt/c/` 사용
